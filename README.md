# ESP32 ECG Monitor

An educational ECG signal acquisition and visualization project using an ESP32 and the AD8232 analog front-end board.

## Overview

This project reads ECG-style biopotential signals from an AD8232 module using an ESP32, then displays the waveform over serial for analysis and visualization.

This is a learning project for signal acquisition, filtering, and heart-rate estimation. It is not a medical device.

## Planned Features

- Real-time ECG waveform capture
- Serial Plotter visualization
- Basic BPM estimation
- Lead-off detection
- Signal filtering
- Optional web dashboard using ESP32 Wi-Fi

## Hardware

- ESP32 development board
- AD8232 ECG front-end board
- 3-lead electrode cable
- Disposable electrode pads
- Breadboard
- Jumper wires
